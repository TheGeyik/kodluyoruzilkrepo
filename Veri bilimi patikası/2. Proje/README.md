## Proje 2 - MERGE SORT

[16,21,11,8,12,22]
1. Yukarı verilen dizinin merge sort türüne göre aşamalarını yazınız.
```
                    [16,21,11,8,12,22]
               [16,21]   [11,8]   [12,22]
        ------------------------------------------
          [16] - [21] - [11]   [8] - [12] - [22]
        ------------------------------------------
               [16,21]   [8,11]   [12,22]
                  [8,11,16,21]   [12,22]
                    [8,11,12,16,21,22]
```

2. Big-O gösterimini yazınız.

- O(nlogn)
